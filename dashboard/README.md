# Purview Workbench Dashboard

Next.js dashboard for the SICO Purview Workbench.

## What this app contains

- Testing tools:
  - Text Extraction
  - Test Data Classification
- Purview Recipes:
  - SIT Library
  - DLP Library
- Build scaffolds:
  - SIT Builder
  - DLP Builder

## Run locally

```bash
cd /Users/toddparker/Library/CloudStorage/OneDrive-KillerCloud/Applications/gits/SICO/dashboard
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```
